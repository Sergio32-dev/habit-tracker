# ✅ Исправлена проблема с автозаполнением

## Проблема

Текст в dropdown автозаполнения браузера ("Сохраненные сведения") сливался с темным фоном, был плохо читаем.

## Решение

Добавлены CSS стили для автозаполнения:
- Белый фон для полей автозаполнения
- Темный текст (#333) для читаемости
- Стили для различных типов автозаполнения (credentials, contacts)

## Что изменено:

В `src/components/AuthScreen.css` добавлены:
- Стили `-webkit-autofill` для полей автозаполнения
- Стили для dropdown списка автозаполнения
- Цвет текста #333 для контраста

---

**Теперь текст в автозаполнении хорошо читается! ✅**
