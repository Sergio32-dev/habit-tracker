# ✅ Исправлена видимость текста в Header для всех тем

## Проблема

На темной теме текст в Header плохо виден, кнопки сливаются с фоном.

## Решение

Добавлены стили для адаптации Header под разные темы:
- Белый текст для всех элементов Header
- Кнопки с прозрачным фоном и белым текстом
- Админ-бейдж с золотистым цветом
- Адаптация под все темы через CSS переменные

## Что изменено:

### 1. Header.css
- Добавлено правило для всех элементов Header: белый цвет текста
- Гарантируется контраст на любом фоне

### 2. themes.css
- Добавлены стили для Header в каждой теме
- Кнопки адаптируются под тему
- Админ-бейдж всегда золотистый
- Текст всегда белый для контраста

---

**Теперь текст в Header хорошо виден на всех темах! ✅**
